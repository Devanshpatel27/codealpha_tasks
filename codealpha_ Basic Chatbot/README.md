## 📜 Project Overview
This is a simple **rule-based, text-based chatbot** developed as part of the **CodeAlpha Python Programming Internship**. The bot runs in the terminal, responds to common phrases and keywords (greetings, small talk, time, date, jokes, help), and exits when the user says goodbye. It demonstrates **`while` loops**, **`if` / `elif` chains**, **string handling**, the **`datetime`** module, and the **`random`** module in Python.

---

## 🧠 Key Python Concepts Used

- `while` loops (continuous conversation until exit)  
- `if` / `elif` / `else` conditions (keyword-style matching)  
- Functions (`def`, `if __name__ == "__main__"`)  
- Lists (joke collection)  
- String handling (`.lower()`, substring checks with `in`)  
- User input / output (`input`, `print`)  
- `datetime` module (current time and date)  
- `random` module (`choice` for random jokes)

---

## ⚙️ How It Works

1. The program starts the **chatbot** and prints a short welcome message.  
2. It enters a loop and waits for **your message** in the terminal.  
3. It checks your text for **known patterns** (for example *hello*, *how are you*, *time*, *date*, *tell me joke*, *help*, *bye*).  
4. For a match, it prints a **fixed or generated reply** (time/date from `datetime`, a random line from a jokes list).  
5. Typing **`help`** shows what kinds of questions the bot understands.  
6. Typing **`bye`** ends the session and the program stops ✅  
7. Unknown input gets a friendly “I don’t understand” message with a hint to use **help** ❓
