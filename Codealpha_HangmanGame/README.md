## 📜 Project Overview

This is a simple **text-based Hangman** game developed as part of the **CodeAlpha Python Programming Internship**. The player guesses a hidden word **one letter at a time** within a limited number of wrong attempts. It demonstrates **loops**, **conditions**, **lists**, **functions**, **string handling**, and the **`random`** module in Python.

---

## 🧠 Key Python Concepts Used

- `if` / `elif` conditions  
- Loops (`while`)  
- Lists and list operations  
- Functions  
- String handling  
- User input / output (`input`, `print`)  
- `random` module (`choice`, `sample`)

---

## ⚙️ How It Works

1. The program selects a **random word** from a predefined list.  
2. **Several letters** from that word are revealed at the start (hint letters), based on word length.  
3. The player guesses **one letter** at a time.  
4. **Correct** guesses show those letters in the word.  
5. **Wrong** guesses increase the wrong-guess count (max **6** wrong guesses).  
6. The game ends when:  
   - The user guesses the full word correctly ✅  
   - **OR** the maximum wrong guesses is reached ❌
